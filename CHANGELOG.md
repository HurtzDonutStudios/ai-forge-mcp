## v1.2.0 — March 26, 2026

**Tier Rename & Pricing Update**

- Renamed subscription tiers to **Solo / Duo / Studio / Enterprise**
- Updated pricing: Solo $79/mo ($35 founder), Duo $139/mo ($65 founder), Studio $249/mo ($149 founder)

- Seat limits: Solo 1, Duo 2, Studio 5, Enterprise custom

## v1.1.0 — March 25, 2026

**Infrastructure & Security**

- **Stripe Integration** — Full purchase-to-license automation: checkout completes, license key is generated and displayed immediately
- **License Validation** — Subscription status is verified automatically; expired licenses are revoked
- **Download System** — Platform auto-detection with download buttons for Windows (~22MB), macOS (~21MB), and Linux (~38MB)
- **Post-Checkout Flow** — Success page displays license key, download buttons, and getting-started instructions
- **Admin Console Security** — Two-factor authentication, account lockout protection, security status dashboard
- **ForgeRoom UI Polish** — SVG social links (Discord, X, Website, GitHub), engine status indicators, visual refinements
- **Version Endpoint** — Deployment verification for license server
- **Database Cleanup** — Purged test data for accurate production analytics

- 250 MCP tools across 10 servers
